Error

Error(string msg)

Deskripsi

Metode ini dipanggil otomatis jika terjadi kesalahan fatal dengan menampilkan pesan kesalahan dan menghentikan eksekusi. Class turunan dapat menimpa metode ini untuk kustomasi penanganan kesalahan akan tetapi HARUS selalu menghentikan eksekusi, jika tidak maka dokumen yang dihasilkan mungkin tidak valid.

Parameter

msg
Pesan kesalahan.
