SetCompression

SetCompression(boolean compress)

Deskripsi

Mengaktifkan atau menonaktifkan kompresi halaman. Jika aktif, maka representasi internal halaman akan dikompres, yang menghasilkan rasio kompresi dokumen sekitar 2.
Kompresi aktif secara default.

Catatan: ekstensi Zlib diperlukan untuk menggunakan fitur ini. Jika tidak ada, maka kompresi akan dinonaktifkan.

Parameter

compress
Gunakan kompresi (true) atau tidak (false).
